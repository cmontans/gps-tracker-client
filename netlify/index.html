💡 Pasos para usar:

  1. Introduce tu nombre
  2. Introduce el nombre del grupo (debe ser igual para todos los participantes)
  3. Activa el GPS en tu móvil
  4. Presiona "Iniciar Seguimiento"
  5. Permite acceso a ubicación cuando te lo pida
  6. Sal al exterior para mejor señal GPS

🔐 Sistema de Grupos:

  • Solo verás usuarios que estén en tu mismo grupo
  • El nombre del grupo distingue mayúsculas/minúsculas
  • Ejemplos: "Amigos", "Carrera2024", "Familia"
  • Todos los participantes deben usar el mismo nombre exacto

⚠️ Importante para GPS continuo:

  • Android: La app mantendrá la pantalla encendida automáticamente. Baja el brillo para ahorrar batería.
  • iOS: Ve a Ajustes → Pantalla → Bloqueo automático → "Nunca" mientras usas la app.
  • Instalar como app: Chrome/Safari → Menú → "Añadir a pantalla de inicio" para mejor rendimiento.

🍎 Usuarios de iPhone/iPad:

  1. Abre esta página en Safari (no Chrome)
  2. Toca el botón "Compartir" (cuadrado con flecha arriba)
  3. Selecciona "Añadir a pantalla de inicio"
  4. Abre la app desde el icono en tu pantalla
  5. Ve a Ajustes → Safari → Ubicación → "Permitir"
  6. Ajustes → Privacidad → Ubicación → Safari → "Mientras se usa"
